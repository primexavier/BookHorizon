## Clone

```bash
git Clone https://github.com/primexavier/BookHorizon.git
```

## First Install

```bash
cd BookHorizon
cd PXLaravel
copy / cp .env.example .env
Php artisan migrate:refresh --seed
```

to Start Project

```bash
cd BookHorizon

```
